Topics

Regular expressions; lexing: scanners, finite automata; parsing: grammars, top-down parsing, bottom-up parsing; abstract syntax trees; type checking; symbol tables; interpreters and virtual machines: executing ASTs, executing bytecode (JIT compilers, binary translation); survey of domain-specific languages; programming paradigms: functional, logic, scripting.

Textbooks

The textbook for the class will be:

Michael L. Scott. Programming Language Pragmatics, Third Edition. Morgan Kaufmann, 2009.
